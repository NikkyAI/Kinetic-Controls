# Kinetic Controls

## Install

add your VCC: [click here](vcc://vpm/addRepo?url=https://nikkyai.github.io/vpm/index.json)
using https://vrc-get.anatawa12.com/en/alcom/ is highly recommended

## Basics

Kinetic Controls provides the framework for building with grappable and pushable faders/levers and similar  
It builds on top of the modular `Driver` classes provided by [Core-Utilities](https://github.com/NikkyAI/Core-Utilities)
as well as the same access and logging utilities for TXL components

## builtin drivers

- FloatSmoothingRateDriver
- IntSmoothingFramesDriver
- TriggerRestFaders
